# Interactive-application-for-managing-timers-and-sequences-built-with-Electron-HTML-CSS-JavaScript-.
An interactive desktop app built with HTML, CSS, and JavaScript, compiled into a standalone Electron application. It enables simultaneous tracking of multiple timers, managing task cycles and sequences, and visually monitoring progress.
All user data and settings are stored locally between sessions, with features like always-on-top mode, sound notifications, and CSV/XLSX label import and export.

Electron / JavaScript / HTML / CSS / LocalStorage / CSV/XLSX import-export
